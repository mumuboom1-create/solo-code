# K446 LUMEA - PbootCMS 企业网站模板

基于 PbootCMS v3.2.5 的高端灯具品牌（LUMEA）企业网站模板。

## 技术栈
- 后端：PHP + PbootCMS v3.2.5 框架
- 数据库：SQLite / MySQL
- 前端：原生 HTML/CSS/JS + Swiper + jQuery

## 功能模块
- 三级栏目导航
- 产品展示（列表 + 详情 + 技术参数）
- 案例展示
- 下载功能
- 招聘功能
- 在线留言
- 联系表单
- 响应式适配

## 目录结构
```
k446-lumea/
├── config/          # 配置文件
├── core/            # 框架内核
├── apps/            # 应用模块（controller + model）
├── template/        # 前台模板
├── skin/            # 静态资源（CSS/JS/图片）
├── static/          # 系统静态资源
├── data/            # SQLite 数据库
└── doc/             # 文档
```
