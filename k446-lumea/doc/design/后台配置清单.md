# 后台配置交付清单

按下面的步骤在 PbootCMS 后台配置即可把新皮肤跑起来，所有配置都可以在后台完成，不用再改模板。

## 一、自定义标签（后台 -> 全局配置 -> 自定义标签）添加以下标签：

| 标签名 | 作用 | 示例内容 |

| `hero_eyebrow` | Hero 小标题 | Atelier · Est. 2006 · Made in China |

## 二、栏目结构（后台 -> 内容管理 -> 栏目管理）

## 三、SEO 基础配置

1. **URL 优化**：后台 -> 全局配置 -> URL 模式 -> 选「伪静态模式」，仓库根目录已经有 .htaccess 支持 Apache，Nginx 需要对应加伪静态规则（PbootCMS 官网有现成规则）。
2. **TKD**：每个栏目、每篇产品/案例后台都有独立「标题/关键词/描述」字段，挨个填即可。模板已经在 head 里自动读取。
3. **图片压缩**：上传前建议用 TinyPNG 离线压缩一遍，保证海外加载速度。
4. **robots.txt**：根目录已经有了，允许 Google 抓取，可以直接用。
