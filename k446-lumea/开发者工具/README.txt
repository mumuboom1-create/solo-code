========================================
  开发者工具 - 授权码生成器
========================================

本文件夹包含授权码生成工具，仅供开发者使用。
部署网站时，不要将此文件夹上传到服务器！

文件说明：
  gen_license.bat  - 双击运行，输入域名即可生成授权码
  gen_license.js   - Node.js 授权码生成脚本（含 RSA 私匙）

使用方法：
  方式一：双击 gen_license.bat，按提示输入客户域名
  方式二：命令行运行 node gen_license.js 客户域名

生成授权码后，让客户填入 config/config.php：
  'sn' => '域名:授权码',

安全提醒：
  - gen_license.js 中包含 RSA 私匙，务必妥善保管
  - 私匙丢失将无法生成新授权码（已有授权码仍有效）
  - 绝不能将此文件夹发给客户或上传到服务器
